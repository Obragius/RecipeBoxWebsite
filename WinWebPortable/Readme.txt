# WinWebPortable for Daniels Zazerskis PHP Website Showcase
# Released from https://github.com/Obragius/RecipeBoxWebsite/releases

## Install
- Unzip the package
- Run the Setup.bat as administrator

## Run
- To run the website open Launch.bat
- Allow access to httpd and mysqld when prompted

## Close
- Close the tab in your browser with the website
- Press the enter key in the command prompt called Launch.bat

## Uninstall
- Delete the zip and folder
- Consider removing the php folder form the system PATH variable
